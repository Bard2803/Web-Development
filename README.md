# Web-Development

### 1. Commerce
Wiki is a auctions page. It has Database created.
It let's the user:
  * Register
  * Create listings
  * Bid on items
  * Add auctions to watchlist
  * Remove auctions from watchlist
  * The owner of the auction to close that auction
  * Comment on the items
  * etc.
#### Technological stack used:
  * Python
  * SQL
  * Django
  * HTML
  * CSS
